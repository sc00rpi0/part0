    activate server

    Note right of browser: The browser executes the callback function that renders the notes

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa
    activate browser
    browser-->>server: [{ "content": "Second exercise", "date": "2023-11-28" }]
    deactivate browser
    activate server
    server-->>browser: [{ "content": "Second exercise", "date": "2023-11-28" }, ... ]
    deactivate server

    Note left of server: The server adds new data to the notes array and sends updated data to browser
